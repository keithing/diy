Weather LCD
-----------

Display the current weather on an external LCD using an Arduino.

### Hardware

1. Arduino UNO
2. LCD screen (16x2)
3. Potentiometer (controls LCD contrast)
4. Resistor (220 ohm)
5. Breadboard
6. Jumper wires
